1. Activity Recognition(DS Fusion)和Activity RecognitionDS
此软件为经过D-S证据理论融合后的行为识别识别结果，一个为特征级融合，一个为决策级融合。

2. Activity Recognition(LDA data fusion)
此软件为经过LDA（线性判别分析）融合后的行为识别识别结果

3. Activity Recognition(No Fusion)
此软件为没有经过数据融合的行为识别识别结果

4. ActivityRecognition(Fusion).sql
此软件为经过数据融合的行为识别程序所用到的数据库文件（mysql）

5. ActivityRecognition(No Fusion).sql
此软件为没有经过数据融合的行为识别程序所用到的数据库文件（mysql）

6. DS_fusion.m
此软件为D-S证据理论的matlab实现代码

7. NLPLibSVM
此软件为SVM（支持向量机）的实现代码
